About This Project

Welcome to Dashboard.io — a simple yet elegant Flask application that started as a “Hello World” project and evolved into an interactive personal dashboard.

  • 🕒 Real-time clock with 12h/24h toggle
  • 🌤 Weather updates using Open-Meteo API
  • 💬 Daily motivational quotes from Quotes API
  • 📝 Personal to-do list saved locally
  • 🌗 Light/Dark theme toggle

This project demonstrates how a basic Flask app can combine backend routes and frontend interactivity to create a fully responsive dashboard — great for learning Flask, API integration, and modern UI design.
